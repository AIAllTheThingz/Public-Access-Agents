# Rust Adoption Checklist

- [ ] Toolchain, edition, MSRV, targets, and features are declared.
- [ ] Cargo lock and dependency strategy are documented.
- [ ] Unsafe and FFI policy is explicit.
- [ ] Formatting, Clippy, test, and build commands are configured.
- [ ] Trust boundaries and secret handling are documented.
- [ ] Completion evidence is integrated and maintainer-reviewed.
