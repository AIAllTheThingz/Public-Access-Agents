# JVM and Runtime Standard

## Requirements

- Declare source, target, and runtime compatibility explicitly.
- Do not rely on locally installed JDK defaults.
- Use immutable data and records where they improve clarity.
- Preserve interruption and cancellation semantics.
- Document memory, garbage-collection, native-library, and container assumptions when operationally relevant.
