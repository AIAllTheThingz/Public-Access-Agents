# Completion Evidence Standard

## Requirements

- List files created, modified, and deleted.
- State behavior, compatibility, security, and operational impact.
- Record exact validation commands and outcomes.
- Identify checks not run and why.
- Describe remaining limitations, assumptions, risks, and follow-up work.

## Evidence

Completion evidence must identify the files and checks that demonstrate these requirements. Exceptions must follow the repository exception process and must not be implied by omission.
