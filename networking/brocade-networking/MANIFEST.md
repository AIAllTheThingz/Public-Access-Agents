# Brocade Networking Package Manifest

## Files

- `AGENTS.md` — mandatory product behavior and evidence rules
- `README.md` — scope, ownership, adoption, lifecycle, validation, recovery, and limitations
- `MANIFEST.md` — package contents and acceptance
- `standards/OPERATIONS_AND_AUTOMATION_STANDARD.md` — implementation and operations controls

## Acceptance

- Current Fabric OS/SANnav and legacy Ethernet/IP ownership boundaries are explicit.
- Fabric/FID, A/B redundancy, zoning, WWPN, endpoint, and configuration ownership is resolved.
- Principal/domain, ISL, MAPS, management, lifecycle, security, firmware, recovery, and actual-state controls are testable.
- Official switch, FOS/SANnav, HBA, and array sources are revalidated for the exact platform and date.
- Examples contain no credentials, real WWPNs, real addresses, or production data.
- Repository validation passes.
