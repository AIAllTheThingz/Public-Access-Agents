# Ubuntu Review Checklist

- [ ] Exact system, Server/Desktop/flavor, release, kernel, architecture and authority
- [ ] Current lifecycle and standard/Pro/ESM package coverage
- [ ] Release-matched APT sources and per-repository signing trust
- [ ] Explicit authorization and transaction/configuration preview
- [ ] Package/Snap/kernel/driver/application compatibility and free space
- [ ] cloud-init and generated-file ownership
- [ ] AppArmor, firewall, identity, SSH and alternate access preserved
- [ ] Netplan/network, services, restart, storage and workload/user impact
- [ ] Backup, restore/rebuild and boot fallback
- [ ] Representative canary, bounded batches and partial-failure handling
- [ ] Supported release-upgrade path and third-party-source disposition
- [ ] Actual state, security, user/workload, backup and monitoring verified
- [ ] Checks not run and residual risk owned
