# RHEL Family Review Checklist

- [ ] Exact host, actual distribution, release, kernel, architecture and management source
- [ ] Current lifecycle, support/subscription and minor-release coverage
- [ ] Distribution-matched repositories and verified signing trust
- [ ] Explicit authorization and transaction preview
- [ ] Package/module/kernel/driver/application compatibility and free space
- [ ] SELinux enforcing, crypto, firewall, audit and Secure Boot preserved
- [ ] Identity, sudo, SSH, SSSD and alternate access protected
- [ ] Services, restart/reboot, network, storage and workload impact
- [ ] Independent backup, restore/rebuild and boot fallback
- [ ] Canary, serial batches, stop conditions and partial-failure handling
- [ ] Distribution-supported upgrade path
- [ ] Actual state, security, workload, backup and monitoring verified
- [ ] Checks not run, residual risk and follow-up owned
