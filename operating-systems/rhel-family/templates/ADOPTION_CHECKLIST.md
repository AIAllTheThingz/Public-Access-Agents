# RHEL Family Adoption Checklist

- [ ] Stable host, environment, owner, actual distribution, release, kernel and architecture recorded
- [ ] Lifecycle, subscription/support, source-review date and management authority recorded
- [ ] Repositories, signing trust, modules, packages and third-party sources inventoried
- [ ] SELinux, crypto policy, firewall, audit, Secure Boot and vulnerability state defined
- [ ] Identity, authselect/SSSD, sudo, SSH, certificates and alternate access mapped
- [ ] Services, network, time, storage, encryption, backup and monitoring inventoried
- [ ] DNF preview, canaries, batches, restarts, stop conditions and recovery defined
- [ ] Distribution-specific major-upgrade policy verified
- [ ] Mixed-repo, signature, lock, dependency, low-space, SELinux, identity, access and boot tests exist
- [ ] Actual kernel/package, security, service, workload, backup and monitoring verification defined
- [ ] Checks not run and residual risk will be reported
