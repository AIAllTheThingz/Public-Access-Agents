# Oracle Linux Adoption Checklist

- [ ] Stable host, owner, release/update, kernel family/version and architecture recorded
- [ ] Premier/Extended/Sustaining phase, entitlement, source-review date and migration owner recorded
- [ ] ULN/yum channels, signing, modules, packages and third-party sources inventoried
- [ ] UEK/RHCK, Ksplice, boot, driver, agent and workload certification reviewed
- [ ] SELinux, firewall, audit, identity/SSH and alternate access defined
- [ ] Services, network, storage, encryption, backup and monitoring defined
- [ ] DNF preview, canaries, batches, Ksplice/reboot decision, stop conditions and recovery defined
- [ ] Supported upgrade path and Oracle workload certification verified
- [ ] Repo, signature, lock, space, kernel, Ksplice, access and boot tests exist
- [ ] Actual release/package/kernel/Ksplice/security/workload/backup/monitoring verification defined
- [ ] Checks not run and residual risk will be reported
