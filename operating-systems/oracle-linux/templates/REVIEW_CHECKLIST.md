# Oracle Linux Review Checklist

- [ ] Exact host, release, UEK/RHCK, architecture and management authority
- [ ] Current support phase, entitlement and security-fix implications
- [ ] Release-matched Oracle channels/repositories and signing trust
- [ ] Explicit authorization and transaction preview
- [ ] Kernel, Ksplice, driver, agent, application and Oracle workload certification
- [ ] SELinux, firewall, audit, identity, SSH and alternate access preserved
- [ ] Services, restart, network, storage and workload impact
- [ ] Independent backup, restore/rebuild and kernel boot fallback
- [ ] Supported upgrade path
- [ ] Canary, bounded batches, stop conditions and partial-failure handling
- [ ] Actual running/effective state, workload, backup and monitoring verified
- [ ] Checks not run and residual risk owned
