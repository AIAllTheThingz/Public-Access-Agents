# Debian Review Checklist

- [ ] Exact host, release/codename/suite, kernel, architecture and management source
- [ ] Supported stable/oldstable/LTS/ELTS status and package coverage
- [ ] Release-matched sources, signing trust and owned pinning
- [ ] Explicit authorization and simulated transaction
- [ ] Essential-package, kernel, firmware, application and space compatibility
- [ ] Security, identity, SSH and alternate access preserved
- [ ] Services, restart, network, storage and workload impact
- [ ] Independent backup, restore/rebuild and boot fallback
- [ ] Representative canary, bounded batches and partial-failure handling
- [ ] Release-note review and supported sequential upgrade path
- [ ] Actual state, workload, backup and monitoring verified
- [ ] Checks not run and residual risk owned
