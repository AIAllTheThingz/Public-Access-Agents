# Windows Server Review Checklist

- [ ] Exact server, domain, environment, role, cluster and management authority
- [ ] Correct edition, release, build, lifecycle phase and support entitlement
- [ ] Explicit authorization and no unrelated mutation
- [ ] Trusted update/policy source and supported interface
- [ ] Application, role, driver, agent, backup and management compatibility
- [ ] Component health, free space, pending restart and conflicting work
- [ ] Alternate access and break-glass ownership
- [ ] Role-aware backup, restore, rollback or rebuild
- [ ] Cluster quorum, drain, failover, capacity and node order where applicable
- [ ] Defender, firewall, BitLocker, audit, credential and baseline preservation
- [ ] Canary, batches, health gates, stop conditions and observation
- [ ] Bounded remoting, retries, timeouts and restarts
- [ ] Negative, partial-failure and recovery tests
- [ ] Actual role/workload, access, security, backup and monitoring verification
- [ ] Checks not run, residual risk and follow-up ownership
