# macOS Review Checklist

- [ ] Exact managed Macs, ownership/custody, hardware, release and MDM authority
- [ ] Current Apple security updates and hardware/tool/application compatibility
- [ ] Explicit authorization, groups, rings and exclusions
- [ ] FileVault escrow and token ownership without key exposure
- [ ] SIP/startup security, Gatekeeper, XProtect, endpoint and firewall preservation
- [ ] PPPC/TCC purpose, privacy, data minimization and user impact
- [ ] Signed/notarized apps/packages/extensions and removal path
- [ ] Identity/certificates, network/VPN, storage, backup and recovery
- [ ] Notification, accessibility, deferral, deadline and restart behavior
- [ ] Offline, eventual-consistency, partial-ring and failed-device handling
- [ ] Separate authority for lock, erase, release and retirement
- [ ] Actual device-side and user/workload verification
- [ ] Checks not run and residual risk owned
