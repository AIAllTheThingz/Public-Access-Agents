# macOS Adoption Checklist

- [ ] Protected stable device identity, ownership/custody, hardware/chip, release/build recorded
- [ ] Apple security-update and hardware compatibility source/review date recorded
- [ ] ABM/ASM, ADE, MDM/declarative, identity, security, app and update authorities mapped
- [ ] FileVault protection/escrow and secure/bootstrap token ownership verified without keys
- [ ] SIP/startup security, Gatekeeper, XProtect, firewall and endpoint controls defined
- [ ] PPPC/TCC, privacy, telemetry, accessibility, user notification and retention reviewed
- [ ] Apps/packages/extensions signing, notarization, identifiers and removal tested
- [ ] Network/VPN, certificates, storage, backup, recovery and Activation Lock handled
- [ ] Rings, deferrals, deadlines, restarts, pause/stop and recovery defined
- [ ] Offline/MDM lag, compatibility, escrow/token, app/extension, restart and re-enrollment tests exist
- [ ] Actual device-side security, encryption, identity, app, backup and user/workload verification defined
- [ ] Destructive actions, checks not run and residual risk are separately recorded
