# SUSE Linux Enterprise Review Checklist

- [ ] Exact host, product, service pack, kernel, architecture and authority
- [ ] Current lifecycle/LTSS and registration
- [ ] Product/module/extension-matched repositories and signing trust
- [ ] Explicit authorization and Zypper solver preview
- [ ] Package/kernel/driver/application compatibility and capacity
- [ ] AppArmor, firewall, audit, identity, SSH and alternate access preserved
- [ ] Btrfs/Snapper space, rollback limits and independent backup
- [ ] Services, HA/SAP sequencing, restart and workload impact
- [ ] Supported migration path and third-party-repository disposition
- [ ] Canary, bounded batches, stop conditions and partial-failure handling
- [ ] Actual state, workload, backup and monitoring verified
- [ ] Checks not run and residual risk owned
