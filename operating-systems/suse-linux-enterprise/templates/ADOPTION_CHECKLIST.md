# SUSE Linux Enterprise Adoption Checklist

- [ ] Stable host, owner, product, service pack, kernel and architecture recorded
- [ ] Lifecycle/LTSS, registration, source-review date and migration owner recorded
- [ ] Products, modules, extensions, repositories, services, keys and patterns inventoried
- [ ] Zypper/RPM, boot, services and pending-restart state inventoried
- [ ] AppArmor, firewall, audit, identity/SSH and alternate access defined
- [ ] Btrfs/Snapper/LVM/encryption, capacity, backup and recovery defined
- [ ] Solver dry-run, canaries, batches, reboots and stop conditions defined
- [ ] Supported service-pack/major migration path and extension compatibility verified
- [ ] Registration, repo/signature, solver, lock, space, snapshot, access and boot tests exist
- [ ] Actual product/package/kernel/security/workload/backup/monitoring verification defined
- [ ] Checks not run and residual risk will be reported
