# Windows Client Review Checklist

- [ ] Exact managed devices, users/custodian classes, rings and exclusions
- [ ] Correct management authority and no co-management conflict
- [ ] Supported Windows 11 release or valid Windows 10 LTSC/ESU/exception
- [ ] Explicit authorization for the exact action
- [ ] Hardware, app, driver, security and management compatibility
- [ ] BitLocker escrow, data protection, backup/sync and recovery
- [ ] Secure Boot, TPM, Defender, firewall, baseline and privilege preservation
- [ ] User notification, accessibility, privacy, deadline and restart impact
- [ ] Canary/ring metrics, pause, stop, rollback and re-enrollment
- [ ] Offline, stale, partial and failed-device handling
- [ ] Distinct approval and data disposition for destructive actions
- [ ] Actual device-side security, application, access and compliance verification
- [ ] Checks not run, residual risk and migration follow-up
