# Project Standards Manifest Example

## Purpose

This directory contains a completed fictitious example of the [Project Standards Manifest template](../PROJECT_MANIFEST_TEMPLATE.json).

## Files

- [`EXAMPLE.json`](EXAMPLE.json)

## Validation

The example contains no template placeholders and is intended to validate against project-manifest schema version `1.1.0`. Empty optional arrays make intentional non-selection explicit; the selected Ubuntu package demonstrates the operating-system field.

## Boundary

The example is not a real project record, authorization, artifact, test run, or approval.
