# Project Profile Decision Matrix

## Purpose

Provides a starting matrix for profile composition decisions.

| Decision | Low | Moderate | High |
|---|---|---|---|
| Profile rationale | brief | recorded | independently reviewed |
| Secondary profiles | as needed | scoped | scoped and reviewed |
| Architecture record | simple | required | detailed and reviewed |
| Threat analysis | if triggered | if triggered | normally required for material trust changes |
| Validation | focused | layered | layered with negative and recovery evidence |
| Nested instructions | optional | recommended for distinct scopes | required for materially different risk boundaries |
| Operational ownership | if deployed | required | independently confirmed |
| Production readiness | if production | required | cross-discipline approval |
| Evidence retention | normal | traceable | durable and auditable |
| Exceptions | owner approval | independent approval | risk-authority approval |

## Decision rule

Profiles scale expected composition. Governance risk classification remains authoritative.
